### üêõ Bug Reports

1. Check [existing issues](https://github.com/swordenkisk/quantum-learning-matrix/issues) first.
2. Open a new issue with:
   - A clear title
   - Steps to reproduce
   - Expected vs. actual behaviour
   - Python / browser version

### üí° Feature Requests

Open an issue with the label **enhancement** and describe:
- The problem you are trying to solve
- Your proposed solution
- Any alternatives you considered

### üîß Pull Requests

1. Fork the repository.
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make your changes.
4. Run the backend with `python backend/server.py` and verify nothing is broken.
5. Open a PR against `main` with a clear description.

## Code Style

- **Python**: Follow PEP 8. Use type hints where practical.
- **JavaScript**: ES2020+, `"use strict"`, 2-space indentation.
- **CSS**: Custom properties for all colours, mobile-first.
